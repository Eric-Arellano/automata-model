# automata-model
CSE 355 project to validate requirements through Finite Automata.

## Prerequisites
1. Haskell Stack

## To install
Assumes Haskell Stack.

1. `stack setup`

## To run
1. `stack build`
1. `stack exec automata-model`

### REPL
1. `stack ghci`
1. `:r` (reload whenever change made)
1. `main` (run main)
